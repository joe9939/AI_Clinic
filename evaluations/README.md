## Report Format

Each report includes:
- `overall.score` — health score 0-100
- `overall.ci_95` — 95% Wilson confidence interval
- `findings` — list of symptomatic symptoms with A/B gap
- `personality` — LLM-generated personality profile
- `total_symptoms`, `asymptomatic`, `symptomatic` — summary counts
